# Case Study Template

## Purpose
Relevant success story from a similar company that demonstrates value and creates peer trust.

### Header

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│  CASE STUDY                                            │
│                                                         │
│  [Title]                                               │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

## Best Practices

1. **Quantify everything** - Use real numbers
2. **Use peer companies** - Similar industry/size
3. **Include a quote** - From the champion
4. **Show the journey** - Pilot → rollout
5. **Make it actionable** - Clear CTA
